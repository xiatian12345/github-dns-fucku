`git clone https://github.com/xiatian12345/github-dns-fucku.git`

### 3.2 执行脚本 
`cd github-dns-fucku`
`node index.js`

#### 如果每次跑到该目录执行index.js文件比较烦可以做一个alias

`alias fuck='node path-to-your-github-fucku/index.js'`
